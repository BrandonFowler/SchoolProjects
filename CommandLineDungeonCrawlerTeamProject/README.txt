This Java project was developed in eclipse. To test compiling and running the source code directly, 
the project folder will need to be imported into eclipse.

To simply play the game, PlayDungeon.exe has been provided

The UML folder contains all UML diagrams that pertain to the project.

The Presentation folder contains our power point for the final presentation. 

The Project Documents folder contains digital copies of the game description
and design patterns documents.
